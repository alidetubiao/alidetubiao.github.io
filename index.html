飞书与slack交互

要实现这个功能，你需要分两部分来处理：一是在飞书上创建一个机器并设置webhook，二是在Slack上创建一个机器并设置webhook。下面是具体的步骤：

飞书机器和webhook设置

  1. 登录飞书开发者平台：https://open.feishu.cn
  2. 创造一个新的应用，并为应该应用添加机器人。
  3. 在应用管理页面，启动事件API，并添加一个事件订阅，设置事件类型为“消息”。
  4. 在你的服务器上，创建一个用于接收飞书事件的API端点。你需要在这个端点中处理飞书发出的消息，并将问题发送到Slack。

Slack 机器和 webhook 设置

  1. 登录 Slack API 页面：https://api.slack.com/
  2. 创建一个新的 Slack App，并为应该应用添加一个机器用户。
  3. 在应用管理页面，启动事件订阅，设置事件类型为“app_mention”。
  4. 在你的服务器上，创建一个用于接收 Slack 事件的 API 端点。你需要在这个端点中处理 Slack 发出的消息，并将问题发送回飞书。

代码示例

以下是一个使用 Python 和 Flask 框架的简单示例，用于发现你的需求

import requests

app = Flask(__name__)

# 飞书机器人的 webhook 地址
FEISHU_WEBHOOK = 'https://open.feishu.cn/open-apis/bot/v2/hook/<your_feishu_webhook_id>'

# Slack 机器人的 token
SLACK_BOT_TOKEN = 'xoxb-<your_slack_bot_token>'

# Slack 机器人的 ID
SLACK_BOT_ID = '<your_slack_bot_id>'

# Slack 频道的 ID
SLACK_CHANNEL_ID = '<your_slack_channel_id>'

@app.route('/feishu_webhook', methods=['POST'])
def feishu_webhook():
    data = request.json
    if 'event' in data:
        event = data['event']
        if 'msg_type' in event and event['msg_type'] == 'text':
            text = event['text_without_at_bot']
            send_question_to_slack(text)
    return jsonify({'Challenge': 'OK'})

@app.route('/slack_webhook', methods=['POST'])
def slack_webhook():
    data = request.json
    if 'event' in data:
        event = data['event']
        if 'type' in event and event['type'] == 'app_mention':
            text = event['text']
            reply_to_feishu(text)
    return jsonify({'Challenge': 'OK'})

def send_question_to_slack(text):
    url = 'https://slack.com/api/chat.postMessage'
    headers = {'Authorization': 'Bearer ' + SLACK_BOT_TOKEN, 'Content-Type': 'application/json'}
    payload = {'channel': SLACK_CHANNEL_ID, 'text': f'<@{SLACK_BOT_ID}> {text}'}
    requests.post(url, headers=headers, json=payload)

def reply_to_feishu(text):
    headers = {'Content-Type': 'application/json'}
    payload = {'msg_type': 'text', 'content': {'text': text}}
    requests.post(FEISHU_WEBHOOK, headers=headers, json=payload)

if __name__ == '__main__':
    app.run()

这个演示示例使用了 Flask 框架创建了两个 API 端点：/feishu_webhook和/slack_webhook，分别用于处理飞书和 Slack 的 webhook 请求。当收到飞书的消息时，它会提问发送给 Slack 中的 Claude 机器人。当收到 Slack 中 Claude 机器人的回答时，它会将回答发送回飞书。
